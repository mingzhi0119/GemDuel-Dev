# Test Governance Matrix

This matrix tracks the negative-path and gate coverage we rely on during release work.

| Area                      | Failure Mode                                                    | Expected Outcome                           | Evidence                                                                                                                       | Status      |
| ------------------------- | --------------------------------------------------------------- | ------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------ | ----------- |
| Reducer command gate      | Invalid coordinates, duplicate selections, out-of-bounds inputs | Reject before mutation                     | `packages/shared/src/logic/__tests__/propertyBoundaries.test.ts`                                                               | `Completed` |
| FSM / phase policy        | Action issued in the wrong phase                                | Deterministic rejection reason             | `packages/shared/src/logic/__tests__/fsmCommandGate.test.ts`, `packages/shared/src/logic/__tests__/securityBoundaries.test.ts` | `Completed` |
| Protocol parser           | Bad protocol version or malformed message                       | Parse or inbound check fails closed        | `packages/shared/src/logic/__tests__/protocolRecoveryMatrix.test.ts`                                                           | `Completed` |
| Replay import             | Invalid replay file or bad action history                       | Replay import is rejected                  | `apps/desktop/src/app/io/__tests__/safeReplayImport.test.ts`, `packages/shared/src/logic/__tests__/replayImport.test.ts`       | `Completed` |
| Desktop governance        | BrowserWindow or preload contract drift                         | `pnpm desktop:check` fails                 | `apps/desktop/electron/__tests__/desktopGovernance.test.ts`, `tools/scripts/check-electron-governance.mjs`                     | `Completed` |
| Release-health governance | Telemetry, docs, or drill drift                                 | `pnpm release:check` fails                 | `tools/scripts/__tests__/releaseHealthChecklist.test.ts`, `tools/scripts/__tests__/releaseHealthOperations.test.ts`            | `Completed` |
| Architecture budgets      | File-size drift or forbidden cross-layer import                 | `pnpm architecture:check` fails            | `tools/scripts/__tests__/architectureBudgets.test.ts`, `tools/scripts/check-architecture-budgets.mjs`                          | `Completed` |
| Bundle budget             | Main renderer chunk exceeds the governed release threshold      | `pnpm bundle:check` fails                  | `tools/scripts/__tests__/buildBudgetReport.test.ts`, `tools/scripts/check-build-budget.mjs`                                    | `Completed` |
| Dependency governance     | Vulnerability, override, SBOM, secret, or env drift             | `pnpm deps:check` fails                    | `tools/scripts/__tests__/dependencyGovernance.test.ts`                                                                         | `Completed` |
| Multiplayer hook recovery | Heartbeat timeout, reconnect, stale host decisions              | Deterministic hook-level recovery behavior | `apps/desktop/src/hooks/__tests__/useConnectionHealth.test.tsx`, `apps/desktop/src/hooks/__tests__/useGameNetwork.test.tsx`    | `Completed` |
