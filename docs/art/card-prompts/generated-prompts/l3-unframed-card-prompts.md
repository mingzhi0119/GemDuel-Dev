# L3 Unframed Card Prompts

This document derives from `docs/art/card-prompts/card-art-prompt-archive.md` and preserves the archived L3 card prompt entries with card-frame generation instructions removed.

Generation invariant for these prompts: produce interior-only 3:4 fantasy card art with no frame, no border, no card-edge trim, no text, no numbers, no UI icons, and no watermark.
