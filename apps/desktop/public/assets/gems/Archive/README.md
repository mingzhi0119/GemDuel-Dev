# Gem Artwork Assets

- `blue.png`, `white.png`, `green.png`, `black.png`, `red.png`, `pearl.png`, `gold.png` 是游戏运行时使用的透明底 gem 素材。
- 这些文件由 `assets/source/single-gems/Part5_SingleGem_*.png` 抠图后导出，源图保留在根级制作素材库 `assets/source/single-gems/`。
- 映射规则：
    - `pearl.png` 对应 `Part5_SingleGem_PinkPearl_01.png`
    - `gold.png` 对应 `Part5_SingleGem_Gold_01.png`
- 以后替换 gem 素材时，保持同名文件即可，不需要再改 UI 代码路径。
