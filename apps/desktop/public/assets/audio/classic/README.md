# GemDuel Classic Sound Effects

Selected runtime sounds are sourced from Kenney's CC0 audio packs:

- `gem-basic.ogg`: Kenney Casino Audio, `Audio/chips-handle-3.ogg`
- `gem-pearl.ogg`: Kenney Interface Sounds, `Audio/glass_004.ogg`
- `card-market.ogg`: Kenney Casino Audio, `Audio/card-slide-3.ogg`
- `card-royal.ogg`: Kenney Interface Sounds, `Audio/confirmation_003.ogg`

Sources:

- https://kenney.nl/assets/casino-audio
- https://kenney.nl/assets/interface-sounds

The source zip files are intentionally not committed.
